Sobre Mim

Desenvolvedor RPA com mais de 4 anos em tecnologia, especializado em automação de processos e no desenvolvimento de soluções que eliminam trabalho manual e devolvem tempo às operações.


Hoje desenvolvo robôs e sistemas que automatizam processos jurídicos, administrativos e de comércio exterior — atuando de ponta a ponta: levantamento de requisitos, arquitetura da solução, desenvolvimento, implantação e sustentação em produção.


O que faço no dia a dia:

  • Robôs de automação (RPA) com Python, C#, Playwright, Selenium e Power Automate Desktop
  • Automação de documentos: geração e processamento de PDF e Word, extração e tratamento de dados
  • Integrações entre sistemas via APIs REST, incluindo sistemas jurídicos como CPJ, XJUS e Preâmbulo
  • Aplicações web e backend em Python (Flask), com MySQL e SQL Server
  • Inteligência Artificial aplicada a processos corporativos (API OpenAI)
  • Dashboards e indicadores (KPIs) para acompanhamento operacional
  • Deploy, servidores e ambientes em cloud (Microsoft Azure)

Antes de migrar para desenvolvimento, passei por suporte e infraestrutura de TI no Tribunal de Justiça do Paraná e na SpaceSet. Essa base faz diferença hoje: entendo o ambiente onde os robôs rodam, e não só o código deles.


Fui escoteiro voluntário por 12 anos, período que moldou a forma como trabalho — organização, disciplina, responsabilidade e a disposição de levar um problema até o fim.


Formado em Gestão da Tecnologia da Informação pela UniCuritiba. Atualmente me especializando em RPA com UiPath e Automate.


Aberto a oportunidades em RPA, automação inteligente e integrações entre sistemas.

Projetos

Robô de Extração de Tribunais

Automação que coleta processos nos tribunais brasileiros: acessa o sistema, faz login, baixa os autos, filtra os documentos relevantes e classifica cada andamento, gerando um JSON estruturado que alimenta o fluxo jurídico seguinte. Arquitetado em pacotes independentes por sistema processual (EPROC, e-SAJ, PJe e PROJUDI) e integrado à API pública DataJud do CNJ. Desenvolvido em equipe.

Python Playwright API DataJud (CNJ) RPA

Robô de Comprovantes Judiciais

Automação de ponta a ponta que faz login no sistema jurídico, exporta o relatório de garantias, resolve o CNPJ correspondente a cada conta e consulta o Comprovante de Resgate de Depósito Judicial no portal público do banco — salvando os PDFs automaticamente, nomeados por processo e período, e gravando o status de cada consulta de volta na base.

Python Playwright PDF Excel

Robôs de Extração de Sinistros e Agenda

Conjunto de automações que extraem históricos de sinistros de portais de seguradoras e a agenda de tarefas do sistema jurídico, convertendo os dados em planilhas e JSON estruturado que servem de entrada para os demais robôs da operação.

Python Playwright JSON openpyxl

Robô de Automação de E-mails (Outlook)

Robô que lê os e-mails de uma pasta específica do Outlook, monta a planilha de controle e a envia automaticamente aos destinatários. Distribuído como executável autossuficiente com interface gráfica e instalador próprio — roda em qualquer máquina Windows sem precisar instalar Python ou dependências.

Python Outlook · pywin32 CustomTkinter PyInstaller

Robô de Deduplicação de Documentos

Automação que varre a base de documentos do sistema jurídico e identifica arquivos duplicados por hash SHA-256, comparando o conteúdo real e não apenas o nome — liberando espaço em disco e evitando retrabalho na gestão documental.

Python SHA-256 Automação de Arquivos

Automação de Documentos de Exportação

Robôs para emissão de DU-E (Declaração Única de Exportação), Certificado de Origem e certificação Halal, preenchendo os sistemas a partir de planilhas de controle e arquivando automaticamente os documentos emitidos. Desenvolvidos para operações de comércio exterior do agronegócio exportador.

Python Playwright RPA Comércio Exterior

Automação de Rotinas de Comércio Exterior

Conjunto de robôs para rotinas operacionais de exportação: controle e conferência de averbação de embarques, preenchimento automático de e-mails a partir de planilhas de controle e impressão em lote de faturas e documentos, com interface gráfica para os operadores.

Python Excel RPA Interface Gráfica

Robô de Cadastro e-SUS (Saúde Pública)

Automação que cadastra fichas de visita domiciliar e territorial no e-SUS PAC a partir de uma planilha de moradores, distribuindo os registros ao longo dos dias úteis — pulando fins de semana e feriados automaticamente — com escolha de turnos, pré-visualização antes da execução e instalador que prepara todo o ambiente sozinho.

Python Playwright Interface Gráfica e-SUS PAC

Automação Jurídica (RPA)

Desenvolvimento de robôs para automação de tarefas jurídicas como emissão de alvarás, geração de guias, cálculos judiciais, petições iniciais e rotinas da controladoria.

Python Selenium Playwright RPA
